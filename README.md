# What a Hero Would Do ⚔️✨

A secure, serverless web application that delivers life advice through distinct fantasy character archetypes. Built to demonstrate secure API key handling, dynamic frontend UI manipulation, and character-prompt engineering using the Anthropic Claude API.

🔗 **Live Demo:** [On-going Progress]

## 🌟 Key Features
- **The Noble Hero:** Selfless, warm, and highly encouraging guidance (Inspired by Himmel).
- **The Ancient Mage:** Quiet, philosophical, and deeply contemplative wisdom (Inspired by Frieren).
- **The Steadfast Warrior:** Blunt, loyal, tough-love, action-oriented advice (Inspired by Eisen).
- **Secure Architecture:** Built using Netlify Serverless Functions to strictly safeguard backend API keys from client-side inspection.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Backend:** Netlify Functions (Serverless Node.js environment)
- **AI Integration:** Anthropic Claude API (`claude-3-5-haiku`)
